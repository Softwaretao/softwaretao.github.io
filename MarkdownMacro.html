Markdown Marco for Confluence


We built this App on the Atlassian-hosted Forge platform so you can enjoy highest levels of security and performance.

A macro which render Markdown content on Confluence page.

You can Try syntax on online Markdown editor.


Markdown is a simple way to format text that looks great on any device.

It doesn’t do anything fancy like change the font size, color, or type — just the essentials, using keyboard symbols you already know.



Add a Markdown Macro to your page

See the Atlassian documentation for adding a macro.

1. When editing Confluence page, select '+' from the toolbar

2. Find the 'Markdown-macro' by name and select it

3. Select the Edit icon to open the configuration panel

4. Configure 'Diagram text' and preview

5. Resume editing the page, and the configuration panel closes

Security tip:

We recommend that you do not present external resources directly in your Markdown content, but instead use third-party URL links.

When you try to access the external resources, a warning message like the one below may appear:

You must be sure that you trust the external website.
