MathJax Macro for Confluence

We built this App on the Atlassian-hosted Forge platform so you can enjoy highest levels of security and performance.

Render math expressions on Confluence pages.

Add a MathJax Macro to your page

See the Atlassian documentation for adding a macro.

  1. When editing Confluence page, select '+' from the toolbar

  2. Find the 'MathJax Macro' by name and select it

  3. Select the Edit icon to open the configuration panel

  4. Configure parameters and preview

  5. Resume editing the page, and the configuration panel closes

Parameters

Parameters are options that you can set to control what and how content from the macro appears on the page. 

Parameter

Description

MathJax Text

If you are not familiar with MathJax, please visit MathJax.org for details.

Type MathJax text in the box. If your MathJax text is embedded in a sentence, surrounding them by '$$', for example: What does $$E=mc^2$$ mean?

Color

Specify the color of the math expression

Text Size

Specify the font size of the math expression
