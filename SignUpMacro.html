Sign Up Macro for Confluence

We built this App on the Atlassian-hosted Forge platform so you can enjoy highest levels of security and performance.

An out of the box macro for team decision-making and communication.

Add a Sign Up Macro to your page

See the Atlassian documentation for adding a macro.

  1. When editing Confluence page, select '+' from the toolbar

  2. Find the 'Sign Up Macro' by name and select it

  3. Select the Edit icon to open the configuration panel

  4. Configure parameters and preview

  5. Resume editing the page, and the configuration panel closes

Parameters

Parameters are options that you can set to control what and how content from the macro appears on the page. 

Parameter

Description

Title

The button's label text.

Fields

The extra information you would like to get. Enter comma separated labels for each extra information field.

Closing Date

Users can only modify information before the closing date.
