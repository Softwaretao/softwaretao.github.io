HTML Macro for Confluence

We built this App on the Atlassian-hosted Forge platform so you can enjoy highest levels of security and performance.

Safely render HTML in Confluence pages. Some dangerous tags will never be rendered.

Add a HTML Macro to your page

See the Atlassian documentation for adding a macro.

  1. When editing Confluence page, select '+' from the toolbar

  2. Find the 'HTML Macro' by name and select it

  3. Select the Edit icon to open the configuration panel

  4. Configure parameters and preview

  5. Resume editing the page, and the configuration panel closes

Parameters

Parameters are options that you can set to control what and how content from the macro appears on the page. 

Parameter

Description

HTML text

The HTML text to safely render. Some dangerous tags will never be rendered.

The following tags are not supported:

applet, base, body, canvas, command, embed, frame, frameset, head, html, link, meta, noscript, object, script, style, title

acronym, area, basefont, bgsound, big, blink, center, col, content, data, datalist, dialog, dir, font, form, hgroup, image, input, isindex, keygen, listing, marquee, menu, menuitem, meter, multicol, nobr, noembed, noframes, optgroup, option, param, plaintext, progress, select, shadow, slot, spacer, strike, template, textarea, tt, wbr, xmp
